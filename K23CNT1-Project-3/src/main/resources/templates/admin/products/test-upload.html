Quay lại danh sách
Hướng dẫn Test:
  1. Chọn Danh mục từ dropdown
  2. Click "Choose File" và chọn ảnh từ máy tính
  3. Xem preview ảnh xuất hiện
  4. Click "Upload và Tạo Sản Phẩm"
  5. Kiểm tra console log và folder src/main/resources/static/images/products/
Console log sẽ hiển thị: 
=== CREATE PRODUCT DEBUG ===
Product name: Test Upload Image
Category ID: [số]
Image file: [tên file]
Uploading image for category: [tên category]
Image uploaded successfully: /images/products/[category]/[uuid].jpg
